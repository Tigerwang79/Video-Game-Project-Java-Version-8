		*** Installation & utilisation du programme : ***



Il faut tout d'abord télécharger l'archive lescorbie_pierre_salihi.zip contenant l'ensemble du projet dont la partie exécutable du jeu se trouvant dans le dossier jeu.
Ensuite à l'intérieur de ce répertoire va se trouver trois fichiers dont deux exécutables :
	-le "Windows_Luncher.bat" prévu pour pouvoir lancer le programme sous l'OS Windows 
En fonction de l'OS sur lequel vous vous trouvez, l'utilisation du programme sera légèrement altérée.



- Utilisation du programme sous l'OS Windows :

Une fois que vous vous trouvez dans le repértoire "Jeu" il suffit simplement de lancer(double-cliquer dessus) "Windows_Luncher.bat".
Le programme s'ouvre dans une console et vous pouvez dès à présent y jouer.



- Utilisation du programme sous l'OS Linux :

Une fois que vous vous trouvez dans le repértoire "Jeu".
Lancez un terminal à partir d'ici (en faisant un clic droit dans le répertoire et sélectionnant l'onglet "Ouvrir dans un terminal"),
puis une fois ouvert vous aurez à écrire en ligne de commande dans la console la commande "java -jar Projet.jar" et de faire entrée.
Ainsi le jeu se lancera dans la console.
